# ESF: Schema Definitions

This directory provides YAML schemas that define the structure of data in Phases 1-3.

| File | Supports Phase | Role |
|---|---|---|
| `attack_schema.yaml` | Phase 1 (Name) | Defines permitted fields and types for attack definitions |

## Planned Additions

| File | Supports Phase | Role |
|---|---|---|
| `ontology_schema.yaml` | Phase 2 (Relate) | Schema for attack chains, trust boundaries, and relationship types |
| `heuristic_schema.yaml` | Phase 3 (Guess) | Schema for detection strategies and response playbooks |
